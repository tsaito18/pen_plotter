# 1. 実験目的
抵抗とコンデンサを組み合わせた CR 直列回路を取り上げ、その基本的な特性、特に周波数に対する電圧伝送比（振幅特性）と位相差（位相特性）を測定することで、交流回路におけるコンデンサの働きに対する理解を深める。
また、本実験を通じてデジタルオシロスコープの Cursor 機能および Measure 機能といった機能に慣れ、得られた測定結果を理論値と比較・検討することで、電気回路学の講義等で学修した内容を確認する。
測定結果の処理として、両対数、片対数、円座標グラフを作成し、測定データのプロットや近似曲線の描画技術も習得する。
# 2. 実験原理
## 2.1. CR 直列回路
抵抗 R とコンデンサ C を直列に接続し、そこに交流電圧 $V$ を加えた回路を CR 直列回路と呼ぶ。直流回路における抵抗と同様に、交流回路では電流の流れにくさをインピーダンスという量で表す。CR 直列回路全体のインピーダンス $Z$ は、抵抗 R の値と、コンデンサの交流に対する抵抗成分であるリアクタンス $X_{C}$​ のベクトル和で与えられる。
インピーダンス $Z$ は、角周波数を $ω$（$ω=2πf$）、虚数単位を $j$ とすると、以下の式 2-1 のように表される。
$$
Z=R+\frac{1}{j\omega C}=R-j\frac{1}{\omega C} \tag{2-1}
$$
インピーダンスの大きさ $|Z|$ は、各成分の二乗和の平方根となり（式 2-2）、電圧と電流の位相差を表す位相角 $ϕ$ は式 2-3 で計算される。
$$
|Z|=\sqrt{ R^2+\left( \frac{1}{\omega C} \right)^2 } \tag{2-2}
$$
$$
ϕ=\tan^{-1}\left( \frac{1}{\omega CR} \right) \tag{2-3}
$$
この回路に流れる電流 $I$ は、交流回路におけるオームの法則 $I=V/Z$ によって求められる。インピーダンス $Z$ が複素数であるため、電流 $I$ も大きさと位相を持つベクトル量となる。具体的には、電流 $I$ の大きさは $∣I∣=∣V∣/∣Z∣$ となり、その位相は電源電圧 $V$ に対して $ϕ$ だけ進むことになる。
また、キルヒホッフの電圧則（KVL）から、回路の状態によらず、電源電圧 $V$ は抵抗 R での電圧降下 $V_{R}$​ とコンデンサ C での電圧降下 $V_{C}$​ の和に等しい。ただし、交流回路ではこれらの電圧の位相が異なるため、ベクトル和（フェーザ和）として考える必要がある。
$$
V=V_{R}+V_{C} \tag{2-4}
$$
各素子にかかる電圧は、$V_{R}=RI$ および $V_{C}​=\frac{I}{jωC}$​ で与えられる。ここで、$V_{R}$​ は電流 $I$ と同相である。一方、$V_{C}$​ は分母の $j$ の影響で、電流 $I$ よりも位相が 90°（$\frac{π}{2}$​ rad）遅れる。この電圧と電流の位相関係は、フェーザ（ベクトル）を用いて図で表すと直感的である。電流を基準のベクトルとすると、$V_{R}$​ は同相であるため同じ向き、$V_{C}$​ は 90° 遅れるため垂直下向きを向くベクトルとなり、その合成ベクトルが電源電圧 $V$ となる。

## 2.2. 高域通過 CR 回路（High-pass filter）
==＜図 2-1: 高域通過 CR 回路（High-pass filter）の一例＞==

高域通過 CR 回路（High-pass filter）は、名前にもある通り、高い周波数の信号は通過させ、低い周波数の信号は減衰させるか遮断する特性を持つフィルタ回路である。図 2-1 に示したように、コンデンサ C と抵抗 R を直列に接続し、入力電圧 $V_{1}$ を与えると、抵抗 R の両端から出力電圧 $V_{2}$ を取り出すことができる。
この回路は、周波数によってコンデンサ C のインピーダンス $Z_{C}$ が変化することによって説明できる。コンデンサのインピーダンスは以下の式 2-5 で与えられる。
$$
Z_{C}=\frac{1}{j\omega C}=\frac{1}{j2\pi fC} \tag{2-5}
$$
ここで、$f$ は入力信号の周波数 \[Hz]、$\omega$ は角周波数（$\omega=2\pi f$）、$C$ は静電容量 \[F]、$j$ は虚数単位である。この式から、インピーダンスの大きさ $|Z_{C}|$ は周波数 $f$ に反比例することが分かる。
入力電圧 $V_{1}$ は、コンデンサ C と抵抗 R で分圧され、出力電圧 $V_{2}$ として出力される。
$$
V_{2}=\frac{R}{R+Z_{C}}V_{1} \tag{2-6}
$$
周波数 $f$ が高い場合、コンデンサのインピーダンス $Z_{C}$ は 0 に近づき、短絡に近い状態となる。その結果、入力電圧 $V_{1}$ の大部分が抵抗 R にかかり、出力電圧 $V_{2}$ は入力電圧 $V_{1}$ とほぼ等しくなる（$V_{2}\approx V_{1}$）。
一方、周波数 $f$ が低い場合、コンデンサのインピーダンス $Z_{C}$ は無限大に近づき、回路が開放されているのに近い状態となる。その結果、入力電圧 $V_{1}$ の大部分がコンデンサ C にかかり、抵抗 R にはほとんど電圧がかからないため、出力電圧 $V_{2}$ は 0 に近づく。
このような原理によって、図 2-1 のような回路は高域通過フィルタとして機能する。なお、出力電圧の大きさが入力電圧の $\frac{1}{\sqrt{ 2 }}$ 倍となる周波数を遮断周波数 $f_{0}$ と呼び、以下の式 2-7 で求められる。
$$
f_{0}=\frac{1}{2\pi CR} \tag{2-7}
$$
遮断周波数を境として、それより高い周波数の信号は通過し、低い周波数の信号は減衰される。

## 2.3. 低域通過 CR 回路（Low-pass filter）
==＜図 2-2: 低域通過 CR 回路（Low-pass filter）の一例＞==

低域通過 CR 回路（Low-pass filter）は、名前にもある通り、低い周波数の信号は通過させ、高い周波数の信号は減衰させるか遮断する特性を持つフィルタ回路である。図 2-2 に示したように、コンデンサ C と抵抗 R を直列に接続し、入力電圧 $V_{1}$ を与えると、コンデンサ C の両端から出力電圧 $V_{2}$ を取り出すことができる。
この回路は、高域通過回路と同様に、周波数によってコンデンサ C のインピーダンス $Z_{C}$ が変化することによって説明できる。コンデンサのインピーダンスは式 2-8 で与えられる。
$$
Z_{C}=\frac{1}{j\omega C}=\frac{1}{j2\pi fC} \tag{2-8}
$$
ここで、$f$ は入力信号の周波数 \[Hz]、$\omega$ は角周波数（$\omega=2\pi f$）、$C$ は静電容量 \[F]、$j$ は虚数単位である。この式から、インピーダンスの大きさ $|Z_{C}|$ は周波数 $f$ に反比例することが分かる。
入力電圧 $V_{1}$ は、コンデンサ C と抵抗 R で分圧され、出力電圧 $V_{2}$ として出力される。
$$
V_{2}=\frac{Z_{C}}{R+Z_{C}}V_{1} \tag{2-9}
$$
周波数 $f$ が低い場合、コンデンサのインピーダンス $Z_{C}$ は無限大に近づき、回路が開放されているのに近い状態となる。その結果、入力電圧 $V_{1}$ の大部分がコンデンサ C にかかり、出力電圧 $V_{2}$ は入力電圧 $V_{1}$ とほぼ等しくなる（$V_{2}\approx V_{1}$）。
一方、周波数 $f$ が高い場合、コンデンサのインピーダンス $Z_{C}$ は 0 に近づき、短絡に近い状態となる。その結果、入力電圧 $V_{1}$ の大部分が抵抗 R にかかり、コンデンサ C にはほとんど電圧がかからないため、出力電圧 $V_{2}$ は 0 に近づく。
このような原理によって、図 2-2 のような回路は低域通過フィルタとして機能する。なお、高域通過回路と同様に、出力電圧の大きさが入力電圧の $\frac{1}{\sqrt{ 2 }}$ 倍となる周波数を遮断周波数 $f_{0}$ と呼び、以下の式 2-10 で求められる。
$$
f_{0}=\frac{1}{2\pi CR} \tag{2-10}
$$
遮断周波数を境として、それより低い周波数の信号は通過し、高い周波数の信号は減衰される。

## 2.4. 位相差の測定法
本実験では、入力電圧 $V_1$​ と出力電圧 $V_2$​ の間の位相差をオシロスコープを用いて測定する。位相差 $ϕ$ は、$V_1$​ と $V_2$​ の波形の時間的なずれ（時間差）$Δt$ \[s] を角度で表したものであり、次式 2-11 で定義される。
$$
ϕ=2π \frac{Δt}{T}​ [rad]=360× \frac{Δt}{T}​ [deg] \tag{2-11}
$$
ここで、$T$ \[s] は $V_1$​ あるいは $V_2$​ の周期である。時間差 $Δt$ および周期 $T$ は、オシロスコープの Cursor 機能を使用して、波形が 0V を横切る点で測定することで、正確な値を得ることができる。$Δt$ は正負の値をとり得るため、それに応じて位相差 $ϕ$ も正負の値となる。
また、デジタルオシロスコープの Measure 機能を用いることで、位相差 $\phi$ を自動で測定し、ディスプレイに表示することも可能である。Measure 機能を利用する際には、オシロスコープの設定において、CH2（出力電圧 $V_2$）を Source とし、Type を Phase、そして Source2 を CH1（入力電圧 $V_1$）と選択することで、二つの波形間の位相差を直接計測できる。この Measure 機能による自動測定は、広範囲にわたる周波数特性を大量に測定する際など、多くのデータを素早く収集する必要がある際に非常に便利である。
Measure 機能を使用して測定を行う際には、その分解能（精度）が画面に表示されている波形の状態に大きく関係するため、電圧軸と時間軸を適切に調整し、画面上に常に 2〜3 周期の波形が表示され、かつ振幅が最大となるように調整することが非常に重要である。しかし、信号が小さくノイズの影響が大きい低周波領域においては、Measure 機能がノイズ成分も含めて電圧を測定してしまうことで誤差が増大する傾向にあるため、信号のレベルが極端に低下する領域においては、カーソル機能などを用いた手作業での測定が必要となる場合もある。

# 3. 実験方法
## 3.1. 遮断周波数における振幅、位相の測定
### 3.1.1. 実験器具
- 低周波発振器（KENWOOD AG-203D）×1
- 同軸ケーブル（低周波発振器用）×1
- デジタルオシロスコープ（Tektronix TDS2002C）×1
- プローブ（オシロスコープ用）×2
- 回路網 BOX ×1
- テーブルタップ ×1
- カーボン抵抗（1.8kΩ、J 級）×1
- メタライズド フィルムコンデンサ（0.047μF、J 級）×1
### 3.1.2. 手順
1) 指定された抵抗 $R$ とコンデンサ $C$ のカラーコードおよび容量値に基づき、遮断周波数 $f_0$ を $f_0 = \frac{1}{2\pi CR}$ の式を用いてあらかじめ計算しておく。

==＜図 3-1: 実験 3.1 で用いる回路＞==

2) 回路網 BOX を用いて、図 3-1 に示すように高域通過 CR 回路（High-pass filter）を配線する。
3) 低周波発振器の出力を回路網 BOX の入力用端子に接続し、入力波形を正弦波に設定する。
4) 低周波発振器の周波数設定を計算で求めた遮断周波数 $f_0$ 付近に設定し、ATTENUATOR（減衰器）を -10 dB に設定する。
5) デジタルオシロの CH1 プローブを入力電圧 $V_1$（端子 1-1' 間）に、CH2 プローブを出力電圧 $V_2$（抵抗 $R$ の両端、端子 2-2' 間）に接続する。
6) オシロの GND およびトリガポイントを画面の中央に設定し、電圧のスケールを約 500 mV/div、時間のスケールを約 100 $\mu$s/div など、適切な値にする。
7) 低周波発振器の AMPLITUDE つまみ（入力電圧設定）を用いて、CH1 に表示される入力電圧 $V_1$ が約 4 $V_{\text{p-p}}$ になるよう調整する。
8) オシロスコープの Cursor 機能を使用して、入力電圧 $V_1$ および出力電圧 $V_2$ のピークピーク値 $V_{1(P-P)}$ と $V_{2(P-P)}$ をそれぞれ計測し、データシートに記録する。
9) 位相差 $\phi$ の測定のため、Cursor 機能を用いて交流波形の周期 $T$ \[$\mu$s] および波形 $V_1$ と $V_2$ の時間的なずれ $\Delta t$ \[$\mu$s] を計測する。このとき、$\Delta t$ の測定精度を向上させるため、時間のスケールを拡大し（時間/div を短く設定）、波形を広げて詳細に読み取るようにする。
10) 測定された $V_{1(P-P)}$ と $V_{2(P-P)}$ の値から電圧伝送比 $\frac{V_2}{V_1}$ を算出し、また、測定された周期 $T$ と時間差 $\Delta t$ の値から位相差 $\phi = 360 \times \frac{\Delta t}{T}$ \[deg] を計算する。
11) 算出した測定値（電圧伝送比および位相差）を、遮断周波数 $f=f_0$ における理論値（電圧伝送比 $\frac{1}{\sqrt{2}} \approx 0.707$、位相差 $45^\circ$）と比較し、誤差を確認する。
## 3.2. 周波数特性の測定
### 3.2.1. 実験器具
- 低周波発振器（KENWOOD AG-203D）×1
- 同軸ケーブル（低周波発振器用）×1
- デジタルオシロスコープ（Tektronix TDS2002C）×1
- プローブ（オシロスコープ用）×2
- 回路網 BOX ×1
- テーブルタップ ×1
- カーボン抵抗（1.8kΩ、J 級）×1
- メタライズド フィルムコンデンサ（0.047μF、J 級）×1
### 3.2.2. 手順
1) 指定された抵抗 $R$ とコンデンサ $C$ のカラーコードおよび容量値に基づき、遮断周波数 $f_0$ を $f_0 = \frac{1}{2\pi CR}$ の式を用いてあらかじめ計算しておく。
2) 回路網 BOX を用いて、図 3-1 に示すように高域通過 CR 回路（High-pass filter）を配線する。
3) デジタルオシロの Measure 機能を用いて、電圧の振幅と位相差を自動で測定するための設定を行う。Measure ボタンを押した後、ディスプレイ右側の選択ボタンを操作し、Measure 1（入力電圧）を Source CH1、Type Pk-Pk に設定する。次に、Measure 2（出力電圧 ）を Source CH2、Type Pk-Pk に設定する。さらに、Measure 3 を Source CH2、Type Phase、Source2 を CH1 と設定し、入力電圧 $V_1$ と出力電圧 $V_R$ 間の位相差 $\phi$ を直接測定できるようにする。
4) 低周波発振器の出力を回路網 BOX の入力用端子に接続し、入力波形を正弦波に設定する。
5) 低周波発振器の周波数設定を計算で求めた遮断周波数 $f_0$ 付近に設定し、ATTENUATOR（減衰器）を -10 dB に設定する。
6) デジタルオシロの CH1 プローブを入力電圧 $V_1$（端子 1-1' 間）に、CH2 プローブを出力電圧 $V_2$（抵抗 $R$ の両端、端子 2-2' 間）に接続する。
7) オシロの GND およびトリガポイントを画面の中央に設定し、電圧のスケールを約 500 mV/div、時間のスケールを約 100 $\mu$s/div など、適切な値にする。
8) 低周波発振器の AMPLITUDE つまみ（入力電圧設定）を用いて、CH1 に表示される入力電圧 $V_1$ が約 4 $V_{\text{p-p}}$ になるよう調整する。
9) まず遮断周波数 $f_0$ に周波数を設定し、その時の入力電圧 $V$、出力電圧 $V_R$、および位相差 $\phi$ を測定しデータシートに記録する。
10) 遮断周波数 $f_0$ から周波数を上げて測定を行う。測定する周波数は、$2f_0$、$4f_0$、$6f_0$、$10f_0$ の各点を設定し、各周波数における $V$、$V_R$、および $\phi$ の測定値をデータシートに記入する。なお、周波数を変化させるたびに、低周波発振器の AMPLITUDE つまみを使用して、CR 回路の入力電圧  が $4 \text{ V}_{p-p}$ になるよう再調整を行う。これは発振器の内部インピーダンスの影響を打ち消すためである。また、設定する入力周波数の調整誤差は $\pm 5\%$ 以下に抑えるよう留意する。
11) 次に、遮断周波数 $f_0$ から周波数を下げて測定を行う。測定する周波数は、$f_0/2$、$f_0/4$、$f_0/6$、$f_0/10$、$f_0/20$、$f_0/40$ の各点を設定し、各周波数における $V$、$V_R$、および $\phi$ の測定値をデータシートに記入する。
12) デジタルオシロの Measure 機能を使用して精度よく測定するため、周波数を変化させるごとに、電圧軸（V/div）と時間軸（time/div）を調整し、画面上に常に 2〜3 周期の波形が表示され、かつ振幅が最大となるように調整する。これは Measure 機能の分解能（精度）に直結するため、忘れずに実施する。
13) 出力電圧 $V_R$ が小さくなり、数値にふらつきが出てくる場合には、おおよその中央値をデータとして記載する。また、データの読み取りは複数回確認し、中央値を採用する（厳密に行う必要はない）。
14) 測定した $V$ と $V_R$ の値から、電圧伝送比 $|V_R/V|$ の計算値を求め、データシートに記入する。
15) 実験終了後、測定した周波数ごとに、電圧比 $|V_R/V|$ と位相差 $\phi$ の理論値を計算し、データシートの対応する欄に記載する。この際、位相差 $\phi$ の計算には、$\phi = \tan^{-1}(f_0/f)$ の関係式を用いる。
16) 最後に、測定データと理論値を基に、電圧伝送比の周波数特性（両対数グラフ）、位相差の周波数特性（片対数グラフ）、およびベクトル軌跡（円座標グラフ）の 3 種類のグラフを作成する。

# 5. 考察
## 5.1. 遮断周波数における振幅、位相の測定
本実験においては、誤差分析の値が電圧伝送比 99.72%、位相 100.8% と、非常に理論値と近い値を示した（精度よく測定できた理由は検討課題にて後述）。具体的には、式 2-7 により求めた遮断周波数 1881 Hz を入力すると、入力電圧 4.00 V の 0.705（約 $\frac{1}{\sqrt{ 2 }}$）倍となる 2.82 V が出力され、これは理論値の 0.707 倍と非常に近い値となった。また、高域通過 CR 回路においては、遮断周波数 1881 Hz を入力すると、位相が 45.34 度進み、これは理論値の 45 度と非常に近い値となった。

本実験においては、高い周波数帯において非常に理論値と近い値を示したが、低い周波数帯においては大きな誤差が生じた。これは出力電圧が小さくなったことによる、相対的なノイズの増大によるものであると考えられる（詳細は検討課題にて後述）。

# 6. 検討課題
## 6.1. 測定値と理論値の比較検討
本実験における計測結果は、デジタルオシロの機能を適切に活用したことにより、ほとんどの周波数において精度の高い（理論値に近い）ものであった。
遮断周波数 $f_{0}$ における電圧振幅および位相差の測定（実験 3.1）においては、デジタルオシロの Cursor 機能を用いて垂直軸（電圧）と水平軸（時間）の計測を行った。位相差 $\phi$ の算出に必要な時間差 $\Delta t$ や周期 $T$ の測定精度を確保するため、時間軸スケールを拡大し（時間/div を短く設定）、波形を広げて時間的なずれを詳細に読み取ることができたことから、時間の読み取り誤差が低減された。また、広範囲の周波数特性を測定した実験（実験 3.2）では、Measure 機能を用いて電圧振幅や位相差を自動で測定した。デジタルオシロの Measure 機能の分解能（精度）は、画面に表示されている波形の状態に大きく依存するため、周波数を変化させる際、電圧軸および時間軸を調整し、画面上に常に 2～3 周期の波形が表示され、かつ振幅が最大となるように調整するようにした。これらの工夫により、測定値は理論値に対して概ね一致した。
一方で、周波数特性の測定結果において、遮断周波数 $f_0$ より極端に低い周波数（例えば $f_0/40$）の領域では、理論値からの誤差が増大する傾向にあった。この低周波領域では、CR 高域通過フィルタである実験回路の特性上、出力電圧 $V_R$ が非常に小さくなる。この結果、オシロスコープの CH2 で測定される $V_R$ の信号に対して、測定環境によるノイズの影響が相対的に大きくなったと考えられる。デジタルオシロの Measure 機能（$V_{p-p}$）は、ノイズも含めた最大振幅を測定してしまうため、信号が小さくノイズの影響が大きい低周波領域において、読み取り誤差が増大し、測定値が理論値よりも大きく算出される要因となったと考えられる。このことから、信号レベルが極端に低下する領域においては、 Measure 機能に頼るだけでなく、Cursor 機能を用いてノイズ成分を無視した精密な測定が必要な場合があることが分かった。
また、今回の実験では、回路素子にカーボン抵抗とメタライズド フィルムコンデンサを用いた。一般に、抵抗やコンデンサは、その抵抗値や容量値にある一定の許容差を持って製造される。今回実験で用いた抵抗およびコンデンサは ±5%（J 級）のものであった。
抵抗やコンデンサに許容差が存在する場合、今回の実験では理論値の基準となる遮断周波数 $f_{0}=\frac{1}{2\pi CR}$ に合計で約 10% 程度の誤差が生じる可能性がある。これによって電圧伝送比や位相特性の値にわずかに差が生じ、特に遮断周波数 $f_{0}$ 付近やそれより低い周波数帯において、測定値と理論値との誤差として現れることが分かっている。
もし、本実験において、今回使用したカーボン抵抗やメタライズド フィルムコンデンサに代わる、より高価で高精度な抵抗やコンデンサを使用した場合、測定特性は理論値に対してより近づくと考えられる。
## 6.2. 低域通過フィルタ回路の CR 回路に関する理論値計算
### 6.2.1. 振幅伝送比 $|V_{2}/V_{1}|$ および $V_{1}$ と $V_{2}$ 間の位相差
低域通過フィルタ回路は、図 2-2 に示すように、抵抗 $R$ とコンデンサ $C$ を直列に接続し、コンデンサ $C$ の両端から出力電圧 $V_{2}$ を取り出す構成を持つ。入力電圧 $V_{1}$ に対する出力電圧 $V_{2}$ の比である電圧伝送比 $\frac{V_{2}}{V_{1}}$ は、コンデンサのインピーダンス $Z_{C} = \frac{1}{j\omega C}$ を用いて、抵抗 $R$ とコンデンサ $C$ による分圧の関係として求められる。
$$ \frac{V_{2}}{V_{1}} = \frac{\frac{1}{j\omega C}}{R+\frac{1}{j\omega C}} = \frac{1}{1+j\omega CR} \tag{6-1} $$
ここで、遮断角周波数 $\omega_{0}$ を $\omega_{0} = \frac{1}{CR}$、遮断周波数 $f_{0}$ を $f_{0} = \frac{1}{2\pi CR}$ と定義し、角周波数 $\omega$ は $\omega = 2\pi f$ の関係にある。これらの定義を用いると、電圧伝送比 $\frac{V_{2}}{V_{1}}$ は周波数 $f$ および $f_{0}$ を用いて次の式 6-2 のように表現される。
$$ \frac{V_{2}}{V_{1}} = \frac{1}{1+j\frac{\omega}{\omega_{0}}} \exp(-j\phi) = \frac{1}{1+j\frac{f}{f_{0}}} \exp(-j\phi) \tag{6-2} $$
この伝達関数から、振幅伝送比 $|V_{2}/V_{1}|$ は、式 6-2 の分母の絶対値の逆数として計算され、以下の式 6-3 で与えられる。
$$ \left|\frac{V_{2}}{V_{1}}\right| = \frac{1}{\sqrt{1 + \left(\frac{\omega}{\omega_{0}}\right)^2}} = \frac{1}{\sqrt{1 + \left(\frac{f}{f_{0}}\right)^2}} \tag{6-3} $$
また、入力電圧 $V_{1}$ と出力電圧 $V_{2}$ 間の位相差 $\phi$ は、式 6-2 の分母の位相角として定義され、以下の式 6-4 で与えられる。
$$ \phi = -\tan^{-1}\left(\frac{\omega}{\omega_{0}}\right) = -\tan^{-1}\left(\frac{f}{f_{0}}\right) \tag{6-4} $$
低域通過フィルタ回路では、出力電圧 $V_{2}$ の位相は入力電圧 $V_{1}$ に対して必ず遅れる（マイナスとなる）特性を持つ。周波数 $f$ が高くなると、位相差 $\phi$ は $0^\circ$ から $-90^\circ$ に近づく。特に、入力周波数 $f$ が遮断周波数 $f_{0}$ に等しい場合、振幅伝送比 $|V_{2}/V_{1}|$ は $\frac{1}{\sqrt{2}} \approx 0.707$ となり、位相差 $\phi$ は $-\tan^{-1}(1) = -45^\circ$ となる。

## 7.2.「2.2. 高域通過 CR 回路（High-pass filter）」の補足
実験原理 2.2 に関して、$V_{1}$ と $V_{2}$ の関係とその位相差に関して補足する。
高域通過 CR 回路の特性は、入力電圧 $V_1$ に対する出力電圧 $V_2$ の比である電圧伝送比 $\frac{V_2}{V_1}$ によって複素数で表される。
入力電圧 $V_1$ と出力電圧 $V_2$ の関係は、コンデンサのインピーダンス $Z_C = \frac{1}{j\omega C}$ と抵抗 $R$ による分圧として得られる。ここで、遮断角周波数 $\omega_0$ を $\omega_0 = \frac{1}{CR}$、遮断周波数 $f_0$ を $f_0 = \frac{1}{2\pi CR}$ とすると、電圧伝送比 $\frac{V_2}{V_1}$ の振幅 $|V_2/V_1|$（電圧伝送比）は式 7-1 で、入力電圧 $V_1$ と出力電圧 $V_2$ 間の位相差 $\phi$ は式 7-2 によって計算される。
$$ \left|\frac{V_{2}}{V_{1}}\right| = \frac{1}{\sqrt{1 + \left(\frac{\omega_0}{\omega}\right)^2}} = \frac{1}{\sqrt{1 + \left(\frac{f_0}{f}\right)^2}} \tag{7-1}$$
$$ \phi = \tan^{-1}\left(\frac{\omega_0}{\omega}\right) = \tan^{-1}\left(\frac{f_0}{f}\right) \tag{7-2}$$
ここで、$\omega=2\pi f$ \[rad/s]、$\omega_0=2\pi f_0$ \[rad/s] である。
電圧伝送比 $|V_2/V_1|$ については、入力周波数 $f$ が遮断周波数 $f_0$ と等しいとき、その値は $\frac{1}{\sqrt{2}}$、すなわち約 0.707 となる。周波数 $f$ が $f_0$ より十分に高い領域では、コンデンサのインピーダンス $Z_C$ が 0 に近づくため、電圧伝送比 $|V_2/V_1|$ は 1 に近づく。一方、周波数 $f$ が $f_0$ より十分に低い領域では、インピーダンス $Z_C$ が無限大に近づき、電圧伝送比 $|V_2/V_1|$ は 0 に近づく。
位相差 $\phi$ については、この高域通過 CR 回路では、出力電圧 $V_2$ の位相は入力電圧 $V_1$ に対して必ず進む（プラスの）位相差を持つ。周波数 $f$ が極めて低い領域では、位相差 $\phi$ は $90^\circ$ に近づき、周波数 $f$ が極めて高い領域では、位相差 $\phi$ は $0^\circ$ に近づく。遮断周波数 $f_0$ のときは、位相差 $\phi$ は $\tan^{-1}(1) = 45^\circ$ となる。
## 7.3.「2.3. 低域通過 CR 回路（Low-pass filter）」の補足
実験原理 2.3 について、$V_{1}$ と $V_{2}$ の関係とその位相差に関して補足する。
低域通過 CR 回路の特性は、入力電圧 $V_1$ に対する出力電圧 $V_2$ の比である電圧伝送比 $\frac{V_2}{V_1}$ によって複素数で表される。
入力電圧 $V_1$ と出力電圧 $V_2$ の関係は、コンデンサのインピーダンス $Z_C = \frac{1}{j\omega C}$ と抵抗 $R$ による分圧として得られる。ここで、遮断角周波数 $\omega_0$ を $\omega_0 = \frac{1}{CR}$、遮断周波数 $f_0$ を $f_0 = \frac{1}{2\pi CR}$ とすると、電圧伝送比 $\frac{V_2}{V_1}$ の振幅 $|V_2/V_1|$（電圧伝送比）は式 7-3 で、入力電圧 $V_1$ と出力電圧 $V_2$ 間の位相差 $\phi$ は式 7-4 によって計算される。
$$ \left|\frac{V_{2}}{V_{1}}\right| = \frac{1}{\sqrt{1 + \left(\frac{\omega}{\omega_0}\right)^2}} = \frac{1}{\sqrt{1 + \left(\frac{f}{f_0}\right)^2}} \tag{7-3}$$
$$ \phi = -\tan^{-1}\left(\frac{\omega}{\omega_0}\right) = -\tan^{-1}\left(\frac{f}{f_0}\right) \tag{7-4}$$
ここで、$\omega=2\pi f$ \[rad/s]、$\omega_0=2\pi f_0$ \[rad/s] である。
電圧伝送比 $|V_2/V_1|$ については、入力周波数 $f$ が遮断周波数 $f_0$ と等しいとき、その値は $\frac{1}{\sqrt{2}}$、すなわち約 0.707 となる。周波数 $f$ が $f_0$ より十分に高い領域では、コンデンサのインピーダンス $Z_C$ が無限大に近づくため、電圧伝送比 $|V_2/V_1|$ は 0 に近づく。一方、周波数 $f$ が $f_0$ より十分に低い領域では、インピーダンス $Z_C$ が 0 に近づき、電圧伝送比 $|V_2/V_1|$ は 1 に近づく。
位相差 $\phi$ については、この低域通過 CR 回路では、出力電圧 $V_2$ の位相は入力電圧 $V_1$ に対して必ず遅れる（マイナスの）位相差を持つ。周波数 $f$ が極めて低い領域では、位相差 $\phi$ は $0^\circ$ に近づき、周波数 $f$ が極めて高い領域では、位相差 $\phi$ は $-90^\circ$ に近づく。遮断周波数 $f_0$ のときは、位相差 $\phi$ は $-\tan^{-1}(1) = -45^\circ$ となる。
